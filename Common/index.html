Overview¶

Current Platform¶

The module variable CurrentPlatform contains a singleton instance of Platform, which abstracts and unifies multiple platform APIs of Python into a single class instance.

Example

example.py

from pyTooling.Common import CurrentPlatform

# Check for a native Linux platform
if CurrentPlatform.IsNativeLinux:
  pass

unittest.py

from pyTooling.Common import CurrentPlatform

class MyTestCase(TestCase):
  @mark.skipif(not CurrentPlatform.IsMinGW64OnWindows, reason="Skipped when platform isn't MinGW64.")
  def test_OnMinGW64(self) -> None:
    pass

Supported platforms

  • Native

    • Linux (x86-64)

    • macOS (x86-64)

    • Windows (x86-64)

  • MSYS2 (on Windows)

    • MSYS

    • Clang64

    • MinGW32

    • MinGW64

    • UCRT64

  • Cygwin

See also

Platform

→ Is... properties describing a platform (and environment) the current Python program is running on.

Helper Functions¶

  • The isnestedclass() functions returns true, if a class is a nested class inside another class.

  • The getsizeof() functions returns the “true” size of a Python object including auxillary data structures like _dict__.

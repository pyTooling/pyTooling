pyTooling.Exceptions¶

A common set of missing exceptions in Python.

Hint

See high-level help for explanations and usage examples.

Exceptions

  • AbstractClassError: The exception is raised, when a class contains methods marked with abstractmethod or mustoverride.

  • MustOverrideClassError: The exception is raised, when a class contains methods marked with must-override.

  • OverloadResolutionError: The exception is raised, when no matching overloaded method was found.

  • ExceptionBase: Base exception derived from Exception for all custom exceptions.

  • EnvironmentException: The exception is raised when an expected environment variable is missing.

  • PlatformNotSupportedException: The exception is raise if the platform is not supported.

  • NotConfiguredException: The exception is raise if the requested setting is not configured.

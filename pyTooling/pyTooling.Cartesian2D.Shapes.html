pyTooling.Cartesian2D.Shapes

An implementation of 2D cartesian shapes for Python.

Classes

  • Shape: Base-class for all 2D cartesian shapes.

  • Trapezium: A Trapezium is a four-sided polygon, having four edges (sides) and four corners (vertices).

  • Rectangle: A rectangle is a trapezium, where opposite edges a parallel to each other and all inner angels are 90°.

  • Square: A square is a rectangle, where all edges have the same length and all inner angels are 90°.

pyTooling.Platform

Common platform information gathered from various sources.

Hint

See high-level help for explanations and usage examples.

Variables

  • CurrentPlatform

Exceptions

  • PlatformException: Base-exception of all exceptions raised by pyTooling.Platform.

  • UnknownPlatformException: The exception is raised by pyTooling.Platform when the platform can’t be determined.

  • UnknownOperatingSystemException: The exception is raised by pyTooling.Platform when the operating system is unknown.

Classes

  • PythonImplementation: An enumeration describing the Python implementation (CPython, PyPy, …).

  • Platforms: A flag describing on which platform Python is running on and/or in which environment it’s running in.

  • Platform: An instance of this class contains all gathered information available from various sources.

The exception is raised by pyTooling.Platform when the platform can’t be determined.

For debugging purposes, a list of system properties from various APIs is added as notes to this exception to ease debugging unknown or new platforms.
